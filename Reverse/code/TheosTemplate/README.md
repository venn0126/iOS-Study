

# 功能介绍：
超级方便，傻瓜式使用
1. 一键创建xcode可以编译theos tweak
2. xcode直接编译tweak
3. 编译、安装一键搞定，不需要额外输入ssh密码

# 使用方式：
## 1. 创建新tweak工程
1. 修改tweakrc.plist中配置
2. cd Bin/
3. 执行 ·sh generate.sh·
4. done

## 2. coding
1. TheosTemplate/Classes目录下直接写OC代码，有代码提示，和原生一样
