## 技能篇
### 如何挑选小龙虾
+ 生长环境，看腹毛和爪毛，人工养殖而且是干净水的是比较干净的，野生养殖的就比较脏
+ 大小，挑选刚长大的，一般是青色的；外壳发黑，铁硬，就是比较老的了，
+ 公，爪子大，腹部有触角，尾巴比较小，母，爪子小，腹部没有触角，尾巴比较大，最好买母虾，肉多，不压秤，
+ 第二对步行足红色是公的，浅色是母的

### 肉食篇
### 孜然牛羊肉卷
+ 食材
	- 牛/羊肉卷
	- 洋葱
	- 孜然粒
+ 准备过程
	- 在锅中倒入冷水，然后大火烧开，下入牛/羊肉卷
	- 煮出肉中的杂质，灯肉变色，捞出备用
	
+ 烹制过程
	- 起锅，烧热，加入宽油进行滑锅，然后在此加入花生油
	- 改为小火，加入孜然粒，炒出香味
	- 改为大火，加入洋葱，炒至透明
	- 加入牛/羊肉，翻炒
	- 转为中火，开始调味，适量盐,老抽，生抽，蚝油，炒40秒
	- 转为大火，加入孜然粉，炒30秒，出锅

### 土豆炖牛腩
+ 食材
	- 牛腩300g
	- 土豆600g
	- 大葱
	- 姜片
	- 八角
	- 桂皮
	- 青花椒
	- 干辣椒
+ 准备过程
	- 土豆削皮切大小均匀的小块，然后冷水浸泡
	- 牛腩切大小均匀的小块
	- 锅中加入冷水，然后倒入牛腩，煮开，撇除浮沫，盛出备用
	
+ 烹制过程
	- 起锅，烧热，加入油，6成热倒入，大葱，姜片，八角，桂皮，干辣椒，炒出香味
	- 改为小火，倒入牛腩，炒3分钟，目的炒干牛肉上的水分
	- 加入一勺豆瓣酱，转中小火，炒出红油
	- 倒入500ml啤酒或者500ml温水，大火煮开，转中小火煮40分钟
	- 开始调味，盐，胡椒粉，白糖，倒入土豆
	- 大火煮口，转小火，盖上锅盖，煮6-10分钟
	- 最后淋上明油，大火爆炒30s，出锅

### 麻辣小龙虾

+ 食材
	- 小龙虾2kg
	- 洋葱，葱，干辣椒，花椒，火锅底料
	- 啤酒500ml
+ 准备过程
	- 去虾线，然后清洗3遍
+ 烹制过程
	- 锅烧热，倒入凉油，下入干辣椒和花椒，葱，炒出香味
	- 倒入火锅底料炒出香味
	- 倒入小龙虾，爆炒，炒至变色
	- 倒入500ml啤酒，大火烧开
	- 转至中小火，继续煮10分钟
	- 之后放入小葱，大火翻炒，出锅

### 蒜蓉小龙虾

+ 食材
	- 小龙虾2kg
	- 大蒜，生姜，小葱胡椒粉，盐，糖
+ 准备过程
	- 去虾线，然后清洗3遍
	- 大蒜250g剁末，生姜40g剁蓉，小葱切碎
	- 准备碗汁，2克胡椒粉，5克的盐，10克糖，加入水稀释，
+ 烹制过程
	- 放入蒸锅，水开蒸6分钟，备用
	- 热锅凉油，4-5成热之后，倒入100克蒜泥，下入姜蓉，适量黄灯笼酱，炒出蒜香味
	- 然后倒入500ml啤酒，再倒入之前准备好的碗汁，大火烧开
	- 倒入小龙虾，盖上盖子，中火煮8分钟，装盘备用

### 清蒸小龙虾

+ 食材
	- 小龙虾2kg
	- 生姜，柠檬，小葱
+ 准备过程
	- 去虾线，然后清洗3遍
	- 放入姜片，小葱，柠檬挤汁，腌制，然后放入冰箱冷藏60分钟
	
+ 烹制过程
	- 水开，蒸8分钟，带着所有的腌制材料蒸
	- 摆盘，出锅

### 啤酒鸭

+ 食材
	- 鸭腿肉2kg
	- 蒜，花椒
	- 郫县豆瓣酱
+ 准备过程
	- 剁鸭腿，然后焯水，洗净

	
+ 烹制过程
	- 热锅凉油，倒入蒜和花椒炒香，然后盛出蒜备用
	- 倒入2勺郫县豆瓣酱，炒出红油，倒入葱和姜，八角，干辣椒和香叶炒香
	- 倒入蚝油炒香，放入鸭肉，翻炒
	- 调味，盐，料酒，白胡椒，生抽，老抽，翻炒
	- 倒入一罐啤酒，转中小火，放入适量盐煮35分钟
	- 开盖，大火收汁，出锅

### 红烧排骨
+ 食材
	- 猪小排2kg

+ 准备过程
	- 排骨冷水下锅，水开煮30s，捞出清洗干净
	- 调料汁，2勺白糖，4勺醋，2勺生抽，2勺蚝油，半勺老抽，一小勺盐搅拌均匀

	
+ 烹制过程
	- 热锅，不加油，倒入排骨干炒
	- 倒入准备的料汁，翻炒
	- 加入开水进去，没过排骨即可，然后加入一块红腐乳
	- 转中火煮40分钟
	- 40分钟后转大火收汁，出锅之前淋入陈醋和芝麻，出锅


### 红烧牛肉
+ 食材
	- 牛腩300g
	- 葱姜蒜花椒，豆瓣酱，干辣椒，八角，香叶

+ 准备过程
	- 牛肉冷水下锅，水开煮30s，捞出清洗干净
	- 鸡蛋煮熟剥壳
	
+ 烹制过程
	- 热锅凉油，葱姜蒜花椒，炒香，加入豆瓣酱，炒香，加入干辣椒，八角和香叶
	- 倒入牛肉，翻炒
	- 调味，蚝油，生抽，老抽，翻炒
	- 锅边加入高度白酒
	- 翻炒加入白水，大火煮开，转中小火
	- 40分钟后，加入盐
	- 加入一些煮好的鸡蛋和鱼豆腐，收汁

### 酱牛肉
+ 食材
	- 牛腱子2kg
	- 糖，老抽，生抽，八角，香叶，桂皮，干辣酱，花椒

+ 准备过程
	- 腱子肉，切拳头大小的块，然后泡入水，期间可以更换水
	- 锅中倒入冷水，倒入牛肉，然后煮开，撇除浮沫，捞出备用

	
+ 烹制过程
	- 热锅凉油，大葱，姜煸出香味
	- 加入清水，放入腱子肉，把糖，老抽，生抽，八角，香叶，桂皮，干辣酱，花椒一起放入
	- 熬煮50分钟，筷子可以扎透的时候，放入2勺豆瓣酱，和盐
	- 不要立即吃，浸一个晚上放冰箱，第二天吃更佳


### 酸汤肥牛
+ 食材
	- 肥牛200g
	- 黄灯笼辣酱100g
	- 金针菇
	- 葱花

+ 准备过程
	- 冷水煮开，倒入肥牛30s，煮出浮沫，捞出备用

	
+ 烹制过程
	- 热锅凉油，放入黄灯笼辣椒酱，炒香，倒入水，加盐，大火煮开
	- 放入金针菇，煮三分钟捞出铺到碗底
	- 放入肥牛煮2分钟
	- 把肉和汤一起倒入碗中
	- 顶部放葱花点缀，最后淋入热油，即可

## 素食篇
### 黄瓜青椒炒鸡蛋
+ 食材
	- 黄瓜/青椒
	- 鸡蛋
+ 准备过程
	- 黄瓜/青椒切片，鸡蛋打匀
	
+ 烹制过程
	- 起锅，烧热，加入多些宽油进行滑锅，然后在此加入花生油
	- 改为小火，倒入鸡蛋，中小火炒40s
	- 盛出备用
	- 再次加入油，7/8成热加入黄瓜/青椒，爆炒1分钟
	- 转为中火，开始调味，适量盐
	- 转为大火，翻炒30秒出锅


### 清炒荷兰豆
+ 食材
	- 荷兰豆
	- 蒜片
+ 准备过程
	- 去头掐尾，然后去除丝
	- 冷水加盐一勺，浸泡5分钟
	- 烧水煮开，加入荷兰豆，40s捞出，放入冷水
	
+ 烹制过程
	- 起锅，烧热，加入多些宽油进行滑锅，然后在此加入花生油
	- 改为小火，倒入蒜片，炒出香味
	- 加入荷兰豆，爆炒1分钟
	- 转为中火，开始调味，适量盐
	- 转为大火，加入明油，翻炒30秒出锅
	
### 炝炒土豆丝
+ 食材
	- 土豆500g
	- 青椒50g
	- 干辣椒，蒜片，花椒
+ 准备过程
	- 土豆切丝，泡水5分钟
	- 锅烧水，水开倒入土豆丝，煮至土豆透明，捞出
	
+ 烹制过程
	- 起锅，烧热，加入适量油，
	- 油6成热，改为小火，倒入干辣椒，蒜片，花椒，炒出香味
	- 倒入土豆丝改大火，爆炒40秒
	- 转为中火，开始调味，适量盐，白糖，生抽沿锅边淋入
	- 转为大火，炒30s，淋入明油，出锅

### 凉拌土豆丝
+ 食材
	- 土豆500g
	- 蒜，芝麻

+ 准备过程
	- 切丝，泡水5分钟
	- 冷水烧开，放入一勺食用油，焯水1分钟，捞出
	- 蒜切碎末
	- 料汁：放入蒜末，芝麻，淋热油，加入生抽，醋，食盐，白糖，香油，搅匀
	
+ 烹制过程
	- 把土豆丝放入到器皿中，淋入料汁，拌匀即可

## 主食篇
### 鸡蛋炒饼
+ 食材
	- 鸡蛋3个
	- 大头菜，饼条

+ 准备过程
	- 鸡蛋搅匀
	- 大头菜切丝

	
+ 烹制过程
	- 热锅凉油，油热，倒入鸡蛋，炒出备用
	- 热锅凉油，放入菜，炒至断生，放入生抽，蚝油，盐，搅拌，放入饼，焖5分钟
	- 倒入鸡蛋
	- 淋入香油和醋，出锅
