# Linux Shell Commands

## Linux命令行快捷键

|  组合键  |                             作用                             |
| :------: | :----------------------------------------------------------: |
| `ctrl+a` |                          跳转到行首                          |
| `ctrl+b` |                       光标左移一个文字                       |
| `ctrl+c` |                         杀死当前进程                         |
| `ctrl+d` |                       删除光标所在文字                       |
| `ctrl+e` |                          跳转到行尾                          |
| `ctrl+f` |                       光标右移一个文字                       |
| `ctrl+h` |                      删除光标前一个文字                      |
| `ctrl+k` |            删除光标后（不包含光标所在）的所有文字            |
| `ctrl+l` |                             清屏                             |
| `ctrl+u` |                         删除所有文字                         |
| `ctrl+w` |            删除光标前（不包含光标所在）的所有文字            |
| `ctrl+y` |                    粘贴或者恢复上次的删除                    |
| `ctrl+z` | 把当前进程转到后台运行，使用`fg`命令恢复。比如`top -d` 然后`ctrl+z` ，到后台，然后`fg`,重新恢复 |
